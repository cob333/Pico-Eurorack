Branches - Bernoulli Gates sketch for 2HPico

Coin‑flip router inspired by Mutable Instruments Branches. 
One trigger in, two mutually‑exclusive outs; probability and level set from the front panel.

Controls
- Button short press: primary mode. Gate mode = Pot1 sets Left/Right probability (center 50/50; left favors RED Out, right favors GREEN Out). Toggle mode = Pot1 sets chance to flip the current side (far left never flips, far right always flips).
- Button long press: secondary mode. Trigger = ~10 ms pulse. Latch = hold high until next flip.
- Pot1: Probability. Pot2: Output level 0–5 V. Pot3/Pot4: unused.

I/O
- Top jack: Trigger In.
- Middle jack: Right Out (right DAC channel).
- Bottom jack: Left Out (left DAC channel).

LED
- Green when Left Out is active, Red when Right Out is active, off when idle. No mode colors.
